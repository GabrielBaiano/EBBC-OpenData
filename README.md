# EBBC-OpenData
EBBC OpenData é uma API pública que disponibiliza metadados das publicações do Encontro Brasileiro de Bibliometria e Cientometria (EBBC), permitindo consultas sobre autores, artigos, instituições, palavras-chave e referências. A plataforma promove Ciência Aberta e apoia análises bibliométricas, cientométricas e informétricas.
