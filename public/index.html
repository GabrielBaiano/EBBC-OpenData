Introdução à API EBBC OpenData

A API do EBBC OpenData fornece acesso direto aos metadados e à curadoria metodológica das publicações do Encontro Brasileiro de Bibliometria e Cientometria (EBBC) das edições de 2012 a 2024.

Com ela, pesquisadores e desenvolvedores podem filtrar artigos por autor, ano, ferramentas de software identificadas no texto, fontes de coleta de dados e etapas metodológicas onde foram aplicadas, além de realizar buscas textuais avançadas nos resumos.

Formato de Resposta

Todos os endpoints retornam dados no formato JSON, usando codificação UTF-8. As requisições de exportação podem retornar também em CSV.

CORS Habilitado

A API suporta requisições de origens cruzadas (CORS), permitindo que você a consulte diretamente de aplicações web front-end sem a necessidade de proxies.

GET

/api/articles

Retorna uma lista paginada e filtrável de artigos.

Parâmetros de Consulta (Query Params)

Parâmetro Tipo Descrição
search string Busca textual no título, resumo, autores, palavras-chave, ferramentas ou fontes.
year string Filtro de edições. Exemplo: 2020 ou 2022,2024.
author string Filtro por nome do autor (pesquisa parcial).
tool string Filtro por software/ferramenta utilizada (ex: R, VOSviewer).
source string Filtro por banco de dados ou fonte de coleta (ex: Scopus, Lattes).
stage string Etapa metodológica (coleta de dados, análise dos dados, visualização).
has_tool boolean Filtra se o artigo utilizou algum software (true ou false).
limit integer Quantidade limite de resultados retornados por página (padrão: 20).
offset integer Quantidade de artigos a pular para paginação (padrão: 0).
sort string Campo para ordenação: title, year, doi (padrão: title).
order string Ordem da ordenação: asc ou desc (padrão: asc).

/api/articles/:doi

Retorna os metadados de um único artigo com base no DOI especificado. O parâmetro :doi pode conter a URL completa do DOI ou apenas o seu sufixo.

GET

/api/articles/export

Exporta os artigos filtrados como arquivo JSON ou CSV para download direto.

Parâmetros de Consulta (Query Params)

Aceita exatamente os mesmos filtros do endpoint /api/articles, com exceção dos parâmetros de paginação (limit e offset).

Parâmetro Tipo Descrição
format string Formato do download: json (padrão) ou csv.
